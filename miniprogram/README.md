# Pour or Fold · 微信小程序版

原生微信小程序(无构建链,微信开发者工具直接打开即可),包含 4 个酒桌游戏:
德州扑克 / 21 点 / 点球大战 / 小姐牌,均与 H5 版逻辑一致。

## 上线步骤

1. **注册小程序账号**(个人主体即可):https://mp.weixin.qq.com → 立即注册 → 小程序。
2. 注册后在「开发管理 → 开发设置」拿到 **AppID**。
3. 下载 **微信开发者工具**(稳定版):https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
4. 开发者工具 → 导入项目 → 选择本目录(`miniprogram/`)→ 填入你的 AppID
   (或直接修改 `project.config.json` 里的 `appid` 字段,当前是占位的 `touristappid`,游客模式也能预览)。
5. 真机预览没问题后 → 工具右上角「上传」→ 到 mp.weixin.qq.com「版本管理」提交审核。

## 类目与审核(重要)

- **不要注册成「小游戏」**:小游戏类目需要游戏备案/版号,个人主体基本走不通。
  本项目按**普通小程序**结构开发,建议选择「工具」类目,定位文案写**"聚会互动工具"**。
- 涉酒内容务必保留页面里的「请理性饮酒/未成年人请勿饮酒」提示;
  描述文案避免出现「赌」「下注赢钱」等词(界面里赌注是"酒",没有真钱,合规)。

## 关于收费(实话)

- **个人主体小程序无法开通微信支付**。想收 1–5 元,需要以**个体工商户/企业**主体注册并开通微信支付商户号。
- 即使开通了支付,**iOS 端微信小程序禁止虚拟商品支付**(苹果规则),只有安卓端可以内购解锁。
- 现实路径:先免费上线攒用户 → 有量后再升级主体接支付(安卓收费 / iOS 免费)。

## 与 H5 版的差异

- 震动反馈用 `wx.vibrateShort/Long` —— **微信内 iPhone 也能震**(H5 做不到的这里补上了)。
- 音效用 `wx.createWebAudioContext` 合成(低版本基础库自动静默降级)。
- 牌面渲染用 WXML 模板(`tpl/cards.wxml`),逻辑层共享 `utils/poker.js`(已用 node 跑过测试)。
